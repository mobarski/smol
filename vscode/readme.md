# Installation

Copy `small-lang` directory into:
    - Linux: `~/.vscode/extensions`
    - macOS: `~/.vscode/extensions`
    - Windows: `%USERPROFILE%\.vscode\extensions`



# Debug

Ctrl+Shift+P -> "Developer: Inspect Editor Tokens and Scopes"
